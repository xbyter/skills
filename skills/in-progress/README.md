# In Progress

仍在开发中的 Skills。它们尚未准备好发布——预期粗糙的边缘、破坏性更改和被放弃的实验。它们被排除在插件和顶层 README 之外，直到它们毕业到稳定桶。

- **[review](./review/SKILL.md)** — 从固定点以来沿两个并行轴审查变更：**Standards**（diff 是否遵循仓库的编码标准？）和 **Spec**（diff 是否忠实实现了源 issue/PRD？）。
- **[writing-beats](./writing-beats/SKILL.md)** — 将文章塑造为 beats 旅程，choose-your-own-adventure 风格。选择起始 beat，只写那个 beat，然后转向下一个，直到文章自然结束。
- **[writing-fragments](./writing-fragments/SKILL.md)** — Grilling session 从你那里挖掘 fragments——异质的写作块——并将它们追加到单一文档作为未来文章的原材料。
- **[writing-shape](./writing-shape/SKILL.md)** — 获取原材料的 markdown 文件，逐段塑造为文章，在每一步争论格式选择。
